========================================================
PASO 1 — Actualizar tu IP en AWS Security Groups
========================================================
  1. Entra en AWS Academy → AWS Console → EC2 → Security Groups → p2-sisdis-sg
  2. Edita la regla de entrada de SSH (puerto 22) y cambia la IP por la tuya actual
  3. Haz lo mismo con la regla del puerto 6443
  4. Arranca las instancias si están paradas (EC2 → Instances → Start)

========================================================
PASO 2 — Clonar el repo en Windows
========================================================
  git clone https://github.com/D1nosXIX/<nombre-del-repo>.git
  cd <nombre-del-repo>

  O si ya lo tienes:
  git pull

========================================================
PASO 3 — Instalar Docker Desktop en Windows (si no lo tienes)
========================================================
  Descargar de: https://www.docker.com/products/docker-desktop/
  Instalar y arrancar Docker Desktop antes de continuar.

========================================================
PASO 4 — Login en Docker Hub
========================================================
  docker login
  (usuario: d1n0s, usar la contraseña de Docker Hub)

========================================================
PASO 5 — Construir las imágenes Docker
========================================================
  Desde la raíz del repo:

  docker build -t d1n0s/p2-broker:latest ./broker/
  docker build -t d1n0s/p2-server:latest ./server/

  Si hay error de "platform", añadir: --platform linux/amd64
  Ejemplo: docker build --platform linux/amd64 -t d1n0s/p2-broker:latest ./broker/

========================================================
PASO 6 — Subir imágenes a Docker Hub
========================================================
  docker push d1n0s/p2-broker:latest
  docker push d1n0s/p2-server:latest

  Verificar en https://hub.docker.com/u/d1n0s que aparecen las dos imágenes.

========================================================

========================================================

  ssh -i labsuser.pem ubuntu@3.82.160.228
